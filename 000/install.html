Install and Setup

Requirements

BeGin works with the following Python packages:

  • pytorch == 1.8.1

  • dgl >= 0.6.1

  • ogb >= 1.3.4

  • torch-scatter >= 2.0.6

  • torch-sparse >= 0.6.9

  • torch-geometric >= 2.0.4

  • dgllife <= 0.3.0

  • rdkit-pypi

For running some examples, you may need the following additional packages:

  • quadprog

  • cvxpy

  • qpth

Installation

You can install BeGin with the following command:

pip install -e .
